# 闪记同步 (Obsidian Flash Sync)

**在微信里记笔记，自动同步到 Obsidian。**

---

## 产品介绍

闪记同步是**闪记助手**的 Obsidian 插件。闪记助手是一款微信小程序，让你在微信里像聊天一样随手记笔记，然后通过本插件一键同步到 Obsidian。

```
微信小程序（记笔记）
    ↓
云服务器（存储）
    ↓
本插件（同步到 Obsidian Vault）
```

---

## 功能

| 功能 | 说明 |
|------|------|
| 配对码连接 | 输入微信小程序上的 6 位码即可绑定，无需复杂配置 |
| 一键同步 | 拉取所有笔记到本地 Vault |
| 自动同步 | 可设置每隔 N 分钟自动同步 |
| 主题分文件 | 一个主题一个 `.md` 文件，新笔记追加写入 |
| 增量同步 | 只同步新增笔记，已同步的自动跳过 |

## 安装

### 从社区插件商店（待上架）
搜索 **闪记同步** 或 **Flash Sync** 安装。

### 手动安装（BRAT）
1. 安装 [BRAT](https://obsidian.md/plugins?id=obsidian42-brat) 插件
2. `BRAT → Add a beta plugin → 输入 henushang/obsidian-flash-sync`
3. 启用 **闪记同步**

### 手动安装
1. 下载 [最新 Release](https://github.com/henushang/obsidian-flash-sync/releases) 中的 `main.js` 和 `manifest.json`
2. 放入 `.obsidian/plugins/obsidian-flash-sync/`
3. 重启 Obsidian，在设置中启用

## 使用

### 第一步：在微信中记笔记

1. 打开微信小程序 **闪记助手**
2. 微信一键登录
3. 创建主题，输入笔记内容（支持文字 + 图片）
4. 首页底部查看 **桌面连接码**

> 小程序还未发布？联系作者获取体验版。

### 第二步：在 Obsidian 中同步

1. 打开 Obsidian 设置 → **闪记同步**
2. 输入微信小程序上的 **6 位配对码**
3. 点击 **连接**（提示"✅ 连接成功"）
4. 点击 **立即同步**
5. 笔记将出现在 Vault 的 `闪记助手/` 目录下

### 可选：自动同步

在插件设置中设置同步间隔（分钟），插件会在后台自动拉取新笔记。

## 同步结构

```
闪记助手/
├── 工作灵感.md         ← 所有工作灵感笔记在一个文件里
├── 读书笔记.md          ← 按时间追加
└── 日记.md
```

每个 `.md` 文件格式：

```markdown
# 工作灵感

> 自动同步自闪记助手

## 2026-06-28 10:30
今天想到一个很好的产品方向...

---

## 2026-06-28 11:00
补充一下具体实施方案...
```

## 微信小程序

闪记助手是一个微信小程序，提供：

- **聊天式记笔记** — 像发微信一样记录灵感
- **图片 OCR** — 拍照自动识别文字
- **主题管理** — 按主题分类整理
- **桌面连接码** — 一键连接 Obsidian

## 技术栈

- **小程序**: 微信原生小程序
- **后端**: Python FastAPI + SQLite
- **插件**: TypeScript + Obsidian API

## 隐私

- 笔记数据存储在用户自己的服务器上
- 插件仅从服务器拉取数据，不收集任何信息
- 不追踪、不分析、不广告
